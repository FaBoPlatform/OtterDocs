組み立て

ベースフレームの組み立て

部材 個数
20x20アルミフレーム 300mm 3本 x 4セット
アルミフレームジョイント 2本 x 4セット
M3イモネジ 4個 x 4セット

ジョイントで、20x20アルミフレーム(300mm)を連結します。

20x20アルミフレーム(300mm)を3本連結し900mmのフレームを接続します。

900mmのフレームを4本作成します。

ベースフレームの連結

部材 個数
20x20アルミフレーム 300mmx3連結 4セット
20x20アルミフレーム 100mm 4本
アルミフレームL字ジョイント 4本
M3イモネジ 16個

300mmx3連結したアルミフレームに、L字ジョイントを2つずつ取り付けます。

100mmのアルミフレームを用いて、300mmx3連結したアルミフレームを連結します。

2セット作成します。

2つのベースフレームの結合

部材 個数
20x20アルミフレームセット 2セット
20x20アルミフレーム 400mm 3本
アルミフレームL字ジョイント 6本
M3イモネジ 12個
M3スペーサーナット 8個

M3スペーサーナットを400mmアルミフレームに3個, 3個, 2個で差し込み、300mmx3連結したアルミフレーム(900mm)を結合します。

L字ジョイントを用いて結合します。

M3スペーサーナットを400mmアルミフレームに3個, 3個, 2個で差し込み、300mmx3連結したアルミフレーム(900mm)を結合します。

土台の固定

部材 個数
カーボン 1個
M3x10ネジ 6個
